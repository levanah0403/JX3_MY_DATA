# 茗伊插件自定義數據集

> 這是《劍網3》茗伊插件集自定義數據項目，通過該項目可在遊戲中快速加載項目中的各種默認數據項。

## 文件結構

 每個文件夾對應插件集的子插件，文件夾內部結構取決於具體實現，詳情可進入子目錄查看幫助。

## 目前支持

 * `MY_TeamMon`： 團隊監控數據。
